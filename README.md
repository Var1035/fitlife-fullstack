## ✨ Overview

**FitLife** is a powerful full-stack fitness application built to provide users with a complete health & wellness ecosystem. From workout tracking and nutrition management to progress analytics and user authentication, FitLife combines modern UI/UX with scalable backend architecture.

Whether you're a beginner starting your fitness journey or a professional athlete tracking performance, FitLife delivers an intuitive and engaging experience.

# ⚡ Installation Guide

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Var1035/fitlife-fullstack.git
cd fitlife-fullstack
```

## 2️⃣ Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the server directory and add the following:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:3000
```

---

# ▶️ Running the Project

## Start Backend Server

```bash
cd server
npm run dev
```

## Start Frontend

```bash
cd client
npm start
```
